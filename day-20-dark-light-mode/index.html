Dark/Light Mode Toggle

Welcome!

This app demonstrates a dark/light mode toggle that persists your preference using localStorage.

Click the button in the header to switch between themes. Your preference will be saved automatically.

Features

  • Toggle between dark and light themes
  • Preference saved to localStorage
  • Theme loads automatically on page refresh
  • Smooth transitions between themes

About localStorage

localStorage is a way to store data on the user's browser that persists even after the page is closed or the browser is restarted. It's perfect for saving user preferences like theme selection.

localStorage.setItem(key, value); localStorage.getItem(key);
